Add user

Enter the following information about the user you wish to add. All fields are mandatory.

What is the primary role for this person? (required)

If none of these roles describe what this person will be doing, you may add a role.
Inspector

This role may:

  • Backdate status changes
  • Enter service request time
  • Manage service requests
  • Submit service requests
  • View invoices

This role may not:

  • Approve remissions
  • Assign service requests
  • Generate invoices
  • View remissions

This role may:

  • Assign service requests
  • Backdate status changes
  • Generate invoices
  • Enter service request time
  • Submit service requests
  • Manage service requests
  • View invoices
  • View remissions

This role may not:

  • Approve/reject invoices
  • Approve remissions
  • View service performance

  • Enter service request time

  • Submit service requests
  • View invoices
  • View remissions
  • View service performance
