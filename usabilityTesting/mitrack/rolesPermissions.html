Roles and permissions

The following roles are in place for this application. To add a new role, press the Add role button, or to modify the permissions for a given role, select the role and press the Edit role button.

Inspector

This role may:

  • Backdate status changes
  • Enter service request time
  • Manage service requests
  • Submit service requests
  • View invoices

This role may not:

  • Approve remissions
  • Assign service requests
  • Generate invoices
  • View remissions
